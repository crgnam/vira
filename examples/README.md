# Examples

These example files is to provide users with clear examples on how to use the Vira API for common use-cases.

These tutorials build upon one another, though it is by no means a requirement to do all (or any) of them.

1. [Vira Box](01_vira_box/README.md) : The basics of creating a scene to be rendered.
2. [Stars](02_stars/README.md) : The basics of using the `vira_tycho2` command line tool, as well as how to render star fields and unresolved objects.
3. [OSIRIS-REx](03_orex/README.md) : The basics of loading geometry and using SPICE to control object placement.
4. [Moon](04_moon/README.md) : Demonstrates using various DEM processing command line tools and rendering images of the moon with dynamic level-of-detail
6. [Python](virapy_examples_/README.md) : Demonstrates how to use the Virapy pythong bindings directly.
